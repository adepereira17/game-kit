# Desafio de Reação Interativo

Este projeto foi desenvolvido como material didático para ensino de **Lógica de Jogos** e **Manipulação Avançada de DOM**.

## Tecnologias e conceitos aplicados
- **HTML semântico:** criação de palco de jogo e áreas de estatísticas.
- **CSS:** Uso de `position: relative` e `absolute` para controle de coordenadas espaciais.
- **JavaScript:** 
  - `Math.random()` para geração de posições imprevisíveis.
  - `setInterval` para controle de cronômetro.
  - Gestão de estados (início, durante e fim do jogo) .
  - Eventos de mouse para captura de cliques.
